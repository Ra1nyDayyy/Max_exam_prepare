马克思刷题网站

打开方式：
1. 双击 open.bat。
2. 或者直接双击 index.html。

功能：
- 已内置当前这份 444 道题的样例题库。
- 支持导入 .json 和 .txt。
- 可以按“导论、第一章……第七章”刷题。
- 左侧有单独入口：“刷题”和“错题本”。
- 错题本支持按当前章节导出 JSON 或 TXT，不会清空浏览器里的错题记录。
- 错题本里的“答案速览”会直接进入错题，并自动显示每一题答案。
- 小键盘或主键盘 1/2/3/4/5 对应 A/B/C/D/E。
- 单选题选中后自动提交；多选题可多次按 1/2/3/4/5 切换，再点“提交答案”。
- 答错的题会自动加入对应章节的错题本，并保存在浏览器本地。

TXT 导入格式示例：
导论
1. 哲学基本问题是（ ）。
A. 实践和认识
B. 思维和存在
C. 主体和客体
D. 现象和本质
答案：B
